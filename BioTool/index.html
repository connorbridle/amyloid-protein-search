About Tool

Amyloid proteins contain short sequences that, in the right conditions or upon mutation, form beta – sheets which aggregate to form rigid fibrils. These proteins and aggregates are associated with a wide range of diseases, most notable neurodegenerative diseases such as Alzheimer’s disease and Parkinson’s disease. This tool uses a pattern motif developed by researches to search through inputted protein sequences and highlight any areas which meet the criterea specified by the Prosite pattern.

  • Navigate the webpage using the navigation options provided
  • To search a single protein, use the "Single Query" input form and click submit
  • To search multiple proteins, use the "Batch Query" input form and enter Uniprot Accession codes separated by new lines
  • Loading times will vary, single inputs should search fast but batch queries may take some time to process
  • The custom regex includes motif overlaps, so all overlaps will be shown in the results
